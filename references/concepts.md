# Concepts

Use this file to choose relevant essays and translate recurring concepts into analysis or action.

## 调查研究

Core texts:

- 《农村调查》的序言和跋
- 《反对本本主义》 if available in future corpus
- 《改造我们的学习》
- 《关于领导方法的若干问题》

Use when the user has insufficient facts, relies on hearsay, over-theorizes, or asks how to understand a situation.

Method:

1. Go downward to the concrete situation.
2. Ask people with direct experience.
3. Compare several cases.
4. Convert findings into a working judgment.
5. Revise judgment after new practice.

## 实践

Core texts:

- 《实践论》
- 《改造我们的学习》

Use when the user asks about knowledge, theory, action, learning, feedback, or validation.

Method:

1. Treat practice as the source and test of knowledge.
2. Move from sensory contact to conceptual judgment.
3. Return judgment to practice.
4. Iterate after success or failure.

## 矛盾

Core texts:

- 《矛盾论》
- 《关于正确处理人民内部矛盾的问题》 if added later

Use when the user asks about complexity, conflict, prioritization, changing situations, or competing forces.

Method:

1. Identify the main contradictions in the situation.
2. Decide which contradiction is principal now.
3. Identify the principal aspect of that contradiction.
4. Note how the hierarchy changes as conditions change.
5. Avoid one-size-fits-all conclusions.

## 主要矛盾

Core texts:

- 《矛盾论》
- 《目前形势和我们的任务》
- 《论政策》

Use when the user is overwhelmed or cannot decide what matters most.

Method:

1. List all pressures.
2. Identify the one that currently determines the rest.
3. Put most effort there.
4. Reassess after the situation changes.

## 群众路线

Core texts:

- 《关于领导方法的若干问题》
- 《组织起来》
- 《在晋绥干部会议上的讲话》
- 《对晋绥日报编辑人员的谈话》

Use when the user asks about communication, organization, leadership, feedback, or mobilizing people.

Method:

1. Gather scattered views from the people involved.
2. Analyze and concentrate them into a clear line.
3. Return the line in a form people can understand and act on.
4. Check whether action produces real participation.

## 统一战线

Core texts:

- 《论反对日本帝国主义的策略》
- 《中国共产党在民族战争中的地位》
- 《统一战线中的独立自主问题》
- 《论政策》
- 《关于目前党的政策中的几个重要问题》

Use when the user faces alliances, stakeholders, conflicting groups, or partial cooperation with disagreement.

Method:

1. Distinguish allies, opponents, middle forces, and unstable actors.
2. Cooperate where interests overlap.
3. Preserve independence on core aims.
4. Avoid treating every disagreement as the same kind of opposition.

## 持久战

Core texts:

- 《论持久战》
- 《抗日游击战争的战略问题》
- 《战争和战略问题》

Use when the user faces long projects, resource asymmetry, uncertainty, or pressure for quick results.

Method:

1. Divide the struggle into stages.
2. Avoid both defeatism and easy-victory thinking.
3. Preserve strength while creating favorable changes.
4. Convert local gains into strategic accumulation.

## 组织起来

Core texts:

- 《组织起来》
- 《关于领导方法的若干问题》
- 《党委会的工作方法》

Use when the problem is fragmented execution, weak coordination, unclear responsibilities, or low participation.

Method:

1. Turn scattered individuals into a coordinated system.
2. Assign concrete responsibilities.
3. Build feedback channels.
4. Review results frequently.

## 党委会的工作方法

Core texts:

- 《党委会的工作方法》
- 《关于建立报告制度》
- 《关于健全党委制》

Use when the user asks about meetings, management, team alignment, decision quality, or leadership habits.

Method:

1. Put issues on the table.
2. Exchange information fully.
3. Avoid one-person decision habits.
4. Make decisions concrete and assign follow-up.
5. Keep reports timely.
