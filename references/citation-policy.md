# Citation Policy

Use this file whenever the user asks for exact wording, source lookup, citation, quote checking, or page/source reliability.

## Rules

- Do not invent direct quotations.
- Search the local corpus before confirming that a phrase appears in 毛选.
- Treat local corpus hits as extracted text, not final scholarly transcription.
- Use article title, volume, and year by default.
- Use page numbers only when the user provides a specific edition or asks to verify against the original PDF.
- If a query does not match the corpus, say that the local corpus did not find it.
- If the wording is close but not exact, call it a near match or paraphrase.
- If the user provides text, quote only from the provided text or from a corpus match.

## Citation Labels

Use these labels when useful:

- `Exact local-corpus match`: wording found in the local Markdown corpus.
- `Near match`: similar wording or concept found, but not exact.
- `Paraphrase`: restated meaning, not original wording.
- `Interpretation`: analytical claim about meaning, structure, or use.
- `Needs verification`: formal citation should be checked against the PDF or edition.

## Recommended Format

```text
Found in: 《实践论》, vol. 1, 1937
Status: Exact local-corpus match; needs PDF verification for formal citation.
Passage: ...
```
